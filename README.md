SmartGrow Pro is a full-stack financial analytics dashboard built using FastAPI, Python, and JavaScript, designed to provide real-time stock market insights and portfolio analysis. The platform integrates live stock price data, portfolio valuation, risk and anomaly detection, technical indicators, PCA-based analysis, and next-day price prediction using machine learning models. It also includes an interactive frontend dashboard and a learning module to help new users understand financial instruments and investment policies. The project is modular, scalable, and suitable for both academic learning and real-world financial data analysis use cases.
